
## 소개

안녕하세요, 저는 강승훈. 커피와 술, WoW와 개발을 좋아합니다.

다른사람들과 함께 소통하고, 즐겁게 지내고 싶어서 개발자가 되었습니다.

단순하고 솔직한 것을 좋아합니다. 

Angular를 주로 사용하며, 다른 프레임워크 또한 공부하는 프론트엔드 개발자 입니다. 

사용자를 배려하는 개발자, 소통하는 개발자가 되고자 합니다.
